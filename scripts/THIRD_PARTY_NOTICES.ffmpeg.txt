This application bundles the FFmpeg tools `ffmpeg` and `ffprobe` as separate
executables, invoked via their command-line interface.

FFmpeg is free software licensed under the GNU General Public License (GPL)
version 2 or later (the bundled static build links GPL components such as
libx264). FFmpeg is a separate program; Jasmine (AGPL-3.0) invokes it over its
command-line interface (mere aggregation under the GPL — the two programs are
not combined into a single work, so each keeps its own license).

  FFmpeg project:  https://ffmpeg.org/
  License:         GNU GPL v2 or later — https://www.gnu.org/licenses/gpl-2.0.html
  Bundled builds:  x86_64 — FFmpeg 8.1.1 static macOS build from evermeet.cx
                            https://evermeet.cx/ffmpeg/
                   arm64  — FFmpeg 8.1 static macOS build from osxexperts.net
                            https://www.osxexperts.net/
                            (ffmpeg sha256 9a08d61f…287028,
                             ffprobe sha256 aab17ac7…8844e6)
  Corresponding source:
                   FFmpeg sources — https://ffmpeg.org/download.html
                   (the exact version above; also obtainable from the FFmpeg git
                   repository at https://git.ffmpeg.org/ffmpeg.git)

Per the GPL, the corresponding source code for the bundled FFmpeg build is
available at the links above. When you change the bundled build (e.g. to an
arm64-native one), update the "Bundled build" line and confirm the new build is
redistributable.
